# ComfyUI 网站

这是 ComfyUI 的官方网站，部署在 GitHub Pages 上。

## 网站功能

- **首页**：显示网站名称和欢迎信息
- **合作联系**：显示联系邮箱 Boss@sometab.com
- **免责声明**：声明本网站与 comfy.org 和 comfy.com 无关

## 访问方式

- GitHub Pages：https://mourom.github.io/comfyui-website
- 自定义域名：http://comfyui.com（需要DNS解析到GitHub Pages）

## 部署说明

网站使用 GitHub Pages 部署，无需备案，全球可访问。
